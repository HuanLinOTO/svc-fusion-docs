# 其他设备

## AMD 显卡用户

默认启动器下载的整合包不支持 A 卡训练和推理，需要先清空 project 文件夹，随后再此处下载[A卡整合包](https://www.123865.com/s/3iQtVv-HVB6d)。适配使用的是 zluda ，可能会有些奇奇怪怪的兼容问题，但是吧，能用就行。

## MAC 用户
